# MERN_AUTH
ystème complet d'authentification avec inscription, connexion, et réinitialisation de mot de passe par OTP.

##  Fonctionnalités

- Inscription d'utilisateur
- Connexion / Déconnexion
- Réinitialisation de mot de passe par email (OTP à 6 chiffres)
- Protection des routes (Frontend + Backend)
- Gestion des tokens avec cookies sécurisés (`httpOnly`)
- Interface moderne avec React + Tailwind CSS
- Notifications avec React Toastify

##  Technologies utilisées

### Frontend
- React.js + Vite
- React Router DOM
- Axios
- Tailwind CSS
- React Toastify


### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT (JSON Web Tokens)
- bcryptjs (hachage des mots de passe)
- Nodemailer (envoi d'emails)
- dotenv
- CORS

  Installation Frontend
  cd client
  npm install

  Installation Backend
  cd server
  npm install
